# node-dlp
Кроссплатформенная система защиты от внутренних угроз (Data Leakage Prevension) для Node.js

Сейчас проект поддерживает контроль следующих каналов утечек:
*Почта
	Агент IMAP4
	Агент POP3
	Агент SMTP - в работе
*Internet
	MITM-proxy
	ICAP-server
	pcap

Отправка уведомлений:
	Email
	Telegram

Хранение данных:
	MongoDB
	PostgresSQL - в работе


Просмотр инцидентов:
	Web
	Telegram
